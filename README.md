> **Note:** This project was developed during my journey in the **ITI Training Program**. It represents a key milestone in mastering **full stack development skills**, **clean architecture & design**, and **production-ready practices**.
   

> **BLOG APP, a full-stack blog platform built with the MERN stack and TypeScript.**
> High-performance frontend, secure authentication, and a robust REST API.

---
 ## 🚀 Key Features               
* **Authentication** 
    Secure user login and registration using **JWT** and **Bcrypt**.
    
* **Image Management** 
    Integrated with **ImageKit** for optimized cloud storage and **Multer**.
    
* **Modern Styling** 
    Built with **Tailwind CSS v4** and **DaisyUI** for a chic, minimalist design.
    
* **Type Safety**
    Full **TypeScript** support on the backend for better maintainability.
    
* **Validation** 
    Data integrity ensured through **class-validator** and **Mongoose** schemas.

---
## 🛠️ Tech Stack                                                             

### 🌐 Frontend
* **Framework:** React 19 (Vite)
* **Routing:** React Router 7
* **Styling:** Tailwind CSS 4, DaisyUI
* **HTTP Client:** Axios
* **Notifications:** React Hot Toast

### ⚙️ Backend
* **Environment:** Node.js, Express 5
* **Language:** TypeScript
* **Database:** MongoDB (via Mongoose)
* **Auth:** JSON Web Tokens (JWT)
* **File Storage:** ImageKit
